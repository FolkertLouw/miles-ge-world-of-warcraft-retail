# Light Leather Market Snapshots

## Purpose

This page will store or link to market observations for [Light Leather](../items/light-leather.md).

The goal is not only to know the current price, but to support decisions such as:

- sell raw now;
- keep for later;
- convert/craft into something else;
- use for profession leveling;
- vendor if AH value is too low.

## Current status

No market data has been imported yet.

## Future snapshot fields

| Field | Meaning |
|---|---|
| region | EU/US/etc. |
| realm_or_auction_house | Realm/connected realm/region market source. |
| observed_at | Timestamp of the price snapshot. |
| market_value | Estimated market price. |
| min_buyout | Cheapest available listing. |
| quantity_available | Supply at time of snapshot. |
| sale_velocity | Estimated speed: instant/fast/normal/slow/very slow. |
| source | Blizzard API, addon scan, manual entry, etc. |

## Recommendation categories

- Quickest value
- Highest potential value
- Safest value
- Vendor floor
- Keep for profession
